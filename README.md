# Dogtor Backend

## Development Instructions

```sh
git clone https://gitlab.lrz.de/seba-master-2021/team-10/backend.git
cd backend
npm install
npm run codegen
npm run dev
```

## Production Instructions

```sh
git clone https://gitlab.lrz.de/seba-master-2021/team-10/backend.git
cd backend
npm install
npm run codegen
npm run build
npm run start
```
